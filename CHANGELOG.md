# Changelog

## v0.1.0 - 2026-04-03

### Added

- Core game loop roguelike/survival chạy trên web 2D (Vite + TypeScript).
- Procedural map generation và fog of war.
- Day/night cycle, weather system và resource respawn.
- Combat hệ projectile, enemy AI thích nghi và pathfinding flow field.
- Base systems: building wall/turret/beacon, crafting medkit.
- Quest board với các nhánh Scavenger / Defender / Hunter.
- Boss encounter: Night Warden với nhiều phase.
- Upgrade bench và stamina loop.
- Relic merchant loop tại căn cứ.
- Minimap, HUD, overlays và save/load local.

### Infrastructure

- GitHub Actions workflow `build.yml` chạy build cho push/pull request vào `main`.
- README nâng cấp theo format portfolio.
